## üé® Style Reference

### Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Dark Navy | #0a0e27 | Backgrounds |
| Bright Cyan | #00d4ff | Accents, icons |
| Deep Purple | #1a0d2e | Secondary background |
| Teal | #00fff5 | Highlights |
| White | #ffffff | Text |
| Light Gray | #e0e0e0 | Secondary text |

### Visual Themes

- **Knowledge Graph:** Connected nodes representing topics
- **Sync Arrows:** Circular arrows showing synchronization
- **Brain/Neural:** Stylized brain or neural network
- **Data Flow:** Streams of data between agents
- **Agent Icons:** Abstract representations of AI agents

### Typography

- Modern sans-serif (Roboto, Inter, SF Pro)
- Bold weights for "KNOWLEDGESYNC"
- Clean, readable at all sizes
